#### 简介

低配版仿追书神器😆

主要实现功能有

* 书架
* 排行版
* 搜索
* 章节缓存、记录阅读位置、换源
* 阅读界面换肤、字体大小调整
* 仿真、左右滑动和无效果三种翻页模式
* 缓存使用了微信[wcdb](https://github.com/Tencent/wcdb)数据库
